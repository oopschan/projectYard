## git init

1. 把当前目录初始化为版本库
2. 当前目录下回生成一个隐藏文件.git

## git add 文件名

1. 把当前目录下的某个文件提交到暂存区
2. git add readme.md 把这个文件提交到暂存区
3. git add . 当前目录所有变动提交到暂存区

## git status

1. 查看当前目录状态

## git commit -m '提交注释'

1. 把暂存区的内容提交到本地仓库

## 本地仓库设计到的三个部分

1. 工作区（实际持有文件）
2. 暂存区
3. 本地仓库

## git log

1. 查看操作日志

## git reflog

1. 查看操作日志（简单版）

## git diff 文件名

1. 查看文件变更信息
